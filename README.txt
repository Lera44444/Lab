Лабораторная 1
Отчет
================================

Назначение программы:
--------------------
Программа предназначена для учета прочитанных книг. Пользователь может добавлять информацию о книгах, просматривать список всех книг, анализировать статистику чтения и сохранять данные в файл.

Функциональные возможности:
--------------------------

1. ДОБАВЛЕНИЕ КНИГИ:
   - Ввод названия книги (до 100 символов)
   - Ввод автора (до 50 символов) 
   - Ввод года прочтения
   - Ввод оценки по 10-балльной шкале
   - Проверка корректности вводимых данных

2. ПРОСМОТР ВСЕХ КНИГ:
   - Отображение списка всех добавленных книг
   - Формат вывода: Название | Автор | Год | Оценка
   - Нумерация книг в списке

3. СТАТИСТИКА:
   - Общее количество прочитанных книг
   - Средняя оценка всех книг
   - Список книг с наивысшей оценкой

4. СОХРАНЕНИЕ ДАННЫХ:
   - Автоматическое сохранение при выходе
   - Формат хранения: текстовый файл books.txt
   - Разделитель полей: символ '$'

5. ЗАГРУЗКА ДАННЫХ:
   - Автоматическая загрузка при запуске программы
   - Восстановление предыдущего состояния

--------------------------

СТРУКТУРА ДАННЫХ:
struct Book {
    char name[101];     // Название книги
    char author[51];    // Автор
    int year;           // Год прочтения  
    float rating;       // Оценка (0-10)
}

ХРАНЕНИЕ ДАННЫХ:
- Массив структур: Book list_book[max_kolich_book]
- Максимальное количество книг: 1000
- Текущее количество: переменная book_count

ФАЙЛОВАЯ СИСТЕМА:
- Имя файла: books.txt
- Формат записи: Название$Автор$Год$Оценка
- Кодировка: текстовый формат

ИСПОЛЬЗУЕМЫЕ БИБЛИОТЕКИ:
#include <iostream>     // Ввод-вывод
#include <fstream>      // Работа с файлами
#include <cstring>      // Работа со строками (char[])
#include <cstdlib>      // Стандартные функции

ОГРАНИЧЕНИЯ ДАННОЙ ПРОГИ:
-----------------------
- Максимальное количество книг: 1000
- Длина названия книги: 100 символов
- Длина имени автора: 50 символов
- Оценка: от 0 до 10 баллов
- Только латинские символы в названиях и авторах

АЛГОРИТМ:
---------------

1. ЗАПУСК ПРОГРАММЫ:
   - Инициализация массива книг
   - Загрузка данных из файла books.txt
   - Отображение главного меню

2. ГЛАВНОЕ МЕНЮ:
   1. Добавить книгу
   2. Показать все книги
   3. Показать статистику
   4. Выход

3. ДОБАВЛЕНИЕ КНИГИ:
   - Проверка на превышение лимита книг
   - Поэтапный ввод данных с проверкой
   - Добавление в массив структур
   - Подтверждение добавления

4. СТАТИСТИКА:
   - Расчет общего количества книг
   - Вычисление средней оценки
   - Поиск книг с максимальной оценкой
   - Форматированный вывод результатов

5. СОХРАНЕНИЕ:
   - Открытие файла для записи
   - Последовательная запись всех книг
   - Использование разделителя '$'
   - Закрытие файла


Конец отчета.
